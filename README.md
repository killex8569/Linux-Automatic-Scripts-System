# Linux-Automatic-Scripts-System
LSAS (Linux Automatic Script Systèm) est un logiciel permettant de pouvoirs géré, créer et déployer des scripts d'automatisation sur linux de manière simple et dynamique


# Features
- Upload de scripts d'automatisations
- Création de scripts d'automatisations
- Modifications et mise en place de scripts définis temporairement (via crontab)
